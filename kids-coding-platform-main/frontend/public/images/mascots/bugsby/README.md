# Bugsby Images

This directory contains all image assets for Bugsby, the coding companion mascot.

## Required Images

### Basic Images
- `static.png` - Main static image of Bugsby (recommended: 160x160px)
- `animated.gif` - Animated version of Bugsby for interactions

### Expression Images
- `happy.png` - Bugsby looking happy and excited
- `excited.png` - Bugsby showing extreme excitement
- `thinking.png` - Bugsby in a thoughtful pose
- `surprised.png` - Bugsby showing surprise
- `celebrating.png` - Bugsby celebrating achievements
- `confused.png` - Bugsby looking puzzled
- `encouraging.png` - Bugsby in an encouraging, supportive pose

## Image Guidelines

- **Format**: PNG for static images, GIF for animations
- **Size**: 160x160px recommended for main images, expressions can be slightly smaller
- **Style**: Should be child-friendly, colorful, and engaging
- **Background**: Transparent background preferred for PNG files
- **Colors**: Should complement the platform's blue (#4A90E2) and yellow (#FFD93D) color scheme
